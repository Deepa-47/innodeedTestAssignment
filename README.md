# Car Showroom
This is a like Car Showroom Website. In which we can see multiple cars and also learn about it.

#Live
https://stunning-twilight-2c1a58.netlify.app

##Feactures
*Responsive
*Slider
*View all details of specific car
*Filter data by car's bodyType
